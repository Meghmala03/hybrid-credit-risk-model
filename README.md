# Hybrid Credit Risk Model

This project builds a 5-factor hybrid credit assessment model using:

- Financial ratios
- PCA
- Logistic Regression
- PD-to-Score transformation
- Qualitative factor integration

## Model Components
- ROC Curve & AUC
- Confusion Matrix
- Correlation Heatmap
- Hybrid Weighted Score
- Risk Bucket Classification

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
